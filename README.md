# Tahoe City Kayak Content Management System

This project is a content management system (CMS) for Tahoe City Kayak, built using [Strapi](https://strapi.io/). Strapi provides a flexible and customizable backend for managing website content, allowing administrators to easily create, update, and organize information such as tours, rentals, blog posts, and more. The CMS is designed to streamline content workflows and support the unique needs of the Tahoe City Kayak website.
